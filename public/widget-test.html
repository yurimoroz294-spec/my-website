Zapi Widget — testovací stránka

Tato stránka simuluje e-shop kde je vložen Zapi chat widget.

Tlačítko chatu naleznete vpravo dole. ↘

Widget je vložen tímto kódem:

<script src="/widget.js" data-api-key="YOUR_KEY" data-greeting="Ahoj! Jak mohu pomoci?"></script>

Poznámka: AI odpovědi vyžadují platný OpenAI API klíč v .env. Bez něj zobrazí chybová hláška "nastala chyba".
